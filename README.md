# GraciePost
A tool I made to let me post artwork to Discord right from my browser. GraciePost fetches a list of Discord channels my bot is in from my server and uses that to create a context menu for images. GraciePost will send json data back to my bot with not only the image url, but also metadata that it picked up from the page like an artist's name or the title of the work, etc.

The icon was designed by [Sasha Koshka](https://github.com/sashakoshka)
